# tello_driver

tello_driver is a ROS2 driver for DJI Tello drones.

## Requirements

Ubuntu 18.04

ROS2 Bouncy

libasio-dev 1.10.8-1

ffmpeg 3.4.4-0ubuntu0.18.04.1, which includes these libraries:
* libavcodec.so.57.107.100
* libavutil.so.55.78.100
* libswscale.so.4.8.100

OpenCV 3
